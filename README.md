# 📧 Spam Email Detector

## 📌 Project Overview
This project classifies emails as *Spam* or *Ham (Not Spam)* using Machine Learning.

## 🚀 Features
- Text preprocessing
- Feature extraction
- Spam/Ham prediction
- Model evaluation

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK (if used)

## 🤖 Machine Learning Model
- Naive Bayes Classifier
